# BMAD Agents and Tasks

BMAD agents are available in Cursor. Use @agent-name (e.g., @dev, @pm, @architect) to invoke them.

## Available Agents
- @analyst - Market research and analysis
- @architect - System architecture design
- @dev - Development implementation
- @pm - Product management
- @po - Product owner
- @qa - Quality assurance
- @sm - Scrum master
- @ux-expert - User experience design
- @bmad-master - Master orchestrator
- @bmad-orchestrator - Workflow orchestrator

## Available Tasks
See individual task files for specific workflows.
