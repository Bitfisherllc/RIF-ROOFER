# Content Rules & Guidelines

This folder contains writing rules and guidelines for RIF website content.

## Available Rulesets

### RIF Recent Projects Rules (v1)
**File:** `rules/rif-recent-projects-rules-v1.md`

Use this for writing "Recently Completed" project posts. Includes:
- SEO optimization guidelines
- Content structure requirements
- Yoast SEO best practices
- Writing style guidelines

## How to Use

When creating content, reference the appropriate ruleset file for:
- SEO requirements
- Content structure
- Writing style
- Formatting guidelines

## Content Types

- **Project Posts** - Recently completed roofing projects
- **Location Pages** - Region, county, city pages
- **Roofer Profiles** - Individual roofer pages
- **Guides** - Educational content about roofing

Each content type may have specific rules and templates.
