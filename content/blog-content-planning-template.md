# Blog Content Planning Template

Use this template when planning individual blog posts to ensure they're optimized for local SEO.

---

## Blog Post Planning Checklist

### Pre-Writing

- [ ] **Category Selected:** Which category does this fit? (See blog-categories-strategy.md)
- [ ] **Primary Keyword:** Main keyword phrase (e.g., "roofers in Miami")
- [ ] **Secondary Keywords:** 3-5 related keywords
- [ ] **Target Location(s):** Which city/county/region is this targeting?
- [ ] **Internal Links Planned:** Which location pages and roofer pages will be linked?

### Content Structure

- [ ] **Title:** Includes location and keyword naturally
- [ ] **Meta Description:** 150-160 chars, includes location and value prop
- [ ] **H1:** Clear, descriptive, includes location if relevant
- [ ] **H2 Sections:** 3-5 main sections planned
- [ ] **Location-Specific Content:** Real local context included (climate, landmarks, codes)
- [ ] **FAQ Section:** 3-5 location-specific questions
- [ ] **Internal Links:** 5-10 links to location pages and roofers
- [ ] **Call-to-Action:** Clear next step for readers

### SEO Elements

- [ ] **URL Slug:** Clean, includes location if relevant
- [ ] **Schema Markup:** FAQ schema, location schema if applicable
- [ ] **Image Alt Text:** Descriptive, includes location if relevant
- [ ] **Internal Linking:** Anchor text includes location names
- [ ] **External Links:** Links to authoritative sources (building codes, FEMA, etc.)

### Quality Check

- [ ] **No Generic Content:** All content is location-specific or Florida-specific
- [ ] **No Keyword Stuffing:** Natural keyword usage
- [ ] **Professional Tone:** Educational, trustworthy, not salesy
- [ ] **Accurate Information:** Verified facts about location, codes, materials
- [ ] **Mobile-Friendly:** Content formatted for mobile reading

## Location-Specific Content Prompts

When writing location-specific content, include:

1. **Climate Context:**
   - Average temperatures
   - Humidity levels
   - Annual rainfall
   - Hurricane frequency/risk
   - Coastal vs. inland considerations

2. **Local Landmarks/Geography:**
   - Nearby bodies of water
   - Elevation considerations
   - Urban vs. suburban vs. rural
   - Historic districts
   - Known weather patterns

3. **Building Code Considerations:**
   - Wind zone requirements
   - Energy code requirements
   - Historic district restrictions
   - HOA requirements (if applicable)

4. **Common Local Issues:**
   - Salt air corrosion (coastal)
   - High heat (inland)
   - Frequent storms (certain regions)
   - Freeze concerns (northern FL)

## Internal Linking Strategy Per Post

Each blog post should link to:

1. **Location Pages (3-5 links):**
   - Primary target location page
   - Related city/county pages mentioned
   - Parent region page
   - Nearby location pages

2. **Roofer Pages (2-3 links):**
   - Roofers serving the target location
   - Preferred roofers when possible
   - Roofers specializing in the topic

3. **Related Blog Posts (2-3 links):**
   - Posts in the same category
   - Complementary topics
   - Seasonal related content

4. **Directory Pages (1-2 links):**
   - Roofer directory filtered by location
   - Service area directory

## Quick Reference: Category to Location Page Mapping

When writing posts in each category, prioritize linking to:

- **Location-Specific Guides** → Target location page + parent pages
- **Storm Damage** → Affected region/county/city pages
- **Material Education** → All service area pages (broad reach)
- **Process Guides** → Target location + roofer directory
- **Seasonal Content** → Region/county pages
- **Cost Guides** → City/county pages
- **Problem-Solving** → Location pages relevant to climate
- **Building Code** → County/city pages (permit offices)
- **Energy Efficiency** → Hot climate city pages
- **Case Studies** → Specific location where project occurred
- **Industry News** → Affected region/county pages
- **Commercial** → Business district/city pages
