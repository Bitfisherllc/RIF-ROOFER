## Quick Theme Switcher

To try different themes, you can modify the Tailwind classes in components. Here are quick changes:

### Switch to Bold & Confident
- Change button backgrounds to `bg-rif-blue-600`
- Add more blue accent sections
- Increase shadow intensity
- Use bolder typography

### Switch to Warm & Approachable
- Use warmer gray tones
- Increase border radius to `rounded-xl`
- Use softer shadows
- Add cream/beige background tones

### Switch to Minimal & Modern
- Remove most shadows
- Increase white space
- Use minimal borders
- Simplify color palette

### Switch to Storm-Ready & Durable
- Use darker blue tones
- Add gray accent backgrounds
- Increase contrast
- Use stronger borders

## Color Usage Examples

### Buttons
```tsx
// Primary button (current)
className="bg-rif-blue-500 hover:bg-rif-blue-600"

// Alternative: Bold
className="bg-rif-blue-600 hover:bg-rif-blue-700"

// Alternative: Light
className="bg-rif-blue-400 hover:bg-rif-blue-500"
```

### Links
```tsx
// Current
className="text-gray-700 hover:text-rif-blue-500"

// Alternative: More prominent
className="text-rif-blue-500 hover:text-rif-blue-600"
```

### Cards
```tsx
// Current: Subtle shadow
className="border border-gray-200 rounded-lg shadow-sm"

// Alternative: More elevation
className="border border-gray-200 rounded-lg shadow-md"

// Alternative: Flat
className="border-2 border-gray-200 rounded-lg"
```
