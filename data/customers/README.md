# Customers Data

This folder contains the list of customers.

## File Location
Place your customer data file(s) in this folder.

## Expected Format
The file can be in any format (CSV, Excel, JSON, etc.). Common formats:
- CSV (`.csv`)
- Excel (`.xlsx`, `.xls`)
- JSON (`.json`)
- Text (`.txt`)

## Usage
Once uploaded, this data will be used to:
- Track customer information
- Manage customer relationships
- Link customers to projects
- Generate reports and analytics

## File Naming
Suggested naming convention:
- `customers.csv`
- `customers-list.xlsx`
- `customers-data.json`

## Privacy Note
Customer data may contain sensitive information. Ensure proper data handling and privacy compliance.
