# Roofers Data

This folder contains the list of all roofers in Florida.

## File Location
Place your roofer data file(s) in this folder.

## Expected Format
The file can be in any format (CSV, Excel, JSON, etc.). Common formats:
- CSV (`.csv`)
- Excel (`.xlsx`, `.xls`)
- JSON (`.json`)
- Text (`.txt`)

## Usage
Once uploaded, this data will be used to:
- Populate the roofer directory
- Match roofers to service areas
- Display roofer profiles
- Manage roofer information in the admin panel

## File Naming
Suggested naming convention:
- `roofers-florida.csv`
- `roofers-list.xlsx`
- `roofers-data.json`
