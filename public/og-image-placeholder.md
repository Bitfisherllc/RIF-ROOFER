# Open Graph Image Setup

## Create OG Image

Create a 1200x630px PNG image for social media sharing.

### Steps:

1. **Export from logo:**
   - Open `/logos/blue-full.svg` in a design tool (Figma, Illustrator, etc.)
   - Set canvas size to 1200x630px
   - Center the logo
   - Add background color if needed (white or blue)
   - Export as PNG

2. **Or create composite:**
   - Use the blue-full logo
   - Add text: "RIF Roofing Installation Framework"
   - Add tagline: "Stone-Coated Metal Roofing in Florida"
   - Export as PNG at 1200x630px

3. **Save as:**
   - `/public/og-image.png`

### Tools:
- Online: Canva, Figma, Adobe Express
- Desktop: Photoshop, Illustrator, GIMP
- Command line: ImageMagick (if installed)
