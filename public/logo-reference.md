# Logo Usage Reference

This folder should contain logo files for the website. For Next.js, place logo files in the `public` directory so they can be referenced as `/logo.svg`.

## Recommended Setup

1. Copy your primary logo files to this `public` folder
2. Use the following naming convention:
   - `logo.svg` - Main logo (blue full version)
   - `logo-white.svg` - White version for dark backgrounds
   - `logo-icon.svg` - Icon only for favicon
   - `favicon.ico` - Browser favicon
   - `og-image.png` - Open Graph social sharing image (1200x630px)

## Current Logo Files

Your logo files are stored in `/logos/` folder. The recommended files to copy to `public/`:

- **Main logo:** `blue-full.svg` → `public/logo.svg`
- **White logo:** `white-full.svg` → `public/logo-white.svg`
- **Icon:** `blue-icon.svg` → `public/logo-icon.svg`
- **Favicon:** Convert `blue-icon.svg` to `favicon.ico`
- **OG Image:** Create 1200x630px PNG from `blue-full.svg` → `public/og-image.png`
